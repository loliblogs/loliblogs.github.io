总结

这篇文章展示了 Markdown 的所有主要语法特性：

  1. 文本格式化：粗体、斜体、删除线、高亮
  2. 结构元素：标题、列表、引用、表格
  3. 代码展示：行内代码、多语言代码块、语法高亮
  4. 数学公式：行内和块级 LaTeX 公式
  5. 链接和图片：各种链接和图片引用方式
  6. 扩展功能：脚注、定义列表、任务列表、Emoji

这些功能足以满足绝大多数技术写作和博客创作的需求。
